# Shauli Daily

גרסת בסיס של עיתון בוקר אישי בעברית, מימין לשמאל, מותאם לטלפון וניתן להתקנה כאפליקציית רשת.

## בדיקה מהירה

אפשר לפתוח את `index.html` לבדיקה חזותית. התקנה מלאה כאפליקציה ועבודה ללא רשת יפעלו לאחר פרסום דרך כתובת מאובטחת.

## פרסום חינמי ב־GitHub Pages

1. יוצרים חשבון ב־github.com. מפעילים אימות דו־שלבי ושומרים את קודי השחזור במקום בטוח.
2. לוחצים על `New repository`, נותנים למאגר את השם `shauli-daily`, בוחרים `Public` כדי להשתמש ב־GitHub Pages ללא תשלום, ולא מוסיפים קובצי התחלה.
3. מעלים למאגר את כל תוכן התיקייה הזאת, כולל התיקייה הנסתרת `.github`.
4. פותחים `Settings` ואז `Pages`. תחת `Source` בוחרים `GitHub Actions`.
5. פותחים את הלשונית `Actions`. התהליך בשם "פרסום האתר" ייתן כתובת קבועה לאחר השלמתו.

## פרסום חינמי ב־Cloudflare Pages

1. יוצרים חשבון ב־cloudflare.com ומפעילים אימות דו־שלבי.
2. באזור `Workers & Pages` בוחרים `Create` ואז `Pages` ו־`Connect to Git`.
3. מחברים את GitHub ומעניקים גישה רק למאגר `shauli-daily`.
4. בוחרים את המאגר. אין צורך בפקודת בנייה; משאירים את שדה פקודת הבנייה ריק ואת תיקיית הפלט בשורש המאגר.
5. לוחצים `Save and Deploy`. מתקבלת כתובת קבועה בסיומת `pages.dev`.

## סודות

אין להכניס סודות לקבצים או לשלוח אותם בצ׳אט. ב־GitHub פותחים `Settings` → `Secrets and variables` → `Actions` ומוסיפים אותם שם בלבד. החיבור ל־Dropbox ול־OpenAI עדיין במצב הכנה ולא יבצע גישה חיצונית עד שנשלים אותו יחד.
